Why we use the Normalize CSS Script

The "Normalize" CSS script accounts for differences in display protocol between browsers. By using it in the CSS file, the webpage will display more uniformly across browsers. This aids in keeping the look of the webpage or website consistent, regarldess of where the viewer is loading the page from.

What is Semantic Markup

Semantic markup is the practice of "marking up" content so it displays in a certain order on a webpage. This can include noting header from body text, and it informs the browser of different kinds of links to display or source content from. Without semantic markup, the browswer would display content as a disorganized mess, if at all.
